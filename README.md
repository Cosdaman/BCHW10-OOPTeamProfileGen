# BCHW10-OOPTeamProfileGen

## Description  

This is a simple JavaScript application which generates a HTML file based on user input through the terminal. It also incorporates testing through JEST.  

## Technologies Used  

- JavaScript
- NodeJS
- Bootstrap
- Inquirer
- Jest
